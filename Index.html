How to Use Scientific Functions:

  • Sine (sin): Input the angle in degrees and then press the sin button. Example: `30 sin`
  • Cosine (cos): Input the angle in degrees and then press the cos button. Example: `60 cos`
  • Tangent (tan): Input the angle in degrees and then press the tan button. Example: `45 tan`
  • Square Root (√): Input the number and then press the √ button. Example: `16 √`
  • Logarithm (log): Input the number and then press the log button. Example: `100 log`
  • Natural Logarithm (ln): Input the number and then press the ln button. Example: `2.718 ln`
  • Exponential (exp): Input the number and then press the exp button. Example: `1 exp`
  • Power (x^y): Input the base, press x^y, then input the exponent. Example: `2 x^y 3 =`
